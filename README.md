# Marcov-Decision-Process

A Markov Decision Process is a mathematical framework used for modeling decision-making in situations where outcomes are partly random and partly controlled by an agent. It consists of states, actions, rewards, and transition probabilities. The key idea is that the next state depends only on the current state and action (Markov property). MDPs are widely used in reinforcement learning to determine the best strategy or policy for maximizing long-term rewards.
